# ADR-001: Выбор облачного провайдера

## Статус

Принято.

## Контекст

Для проекта требуется облачная инфраструктура: Object Storage, Kafka-compatible broker и VM для запуска Docker-сервисов.

## Решение

В качестве целевого облачного провайдера выбран Yandex Cloud, так как он предоставляет Object Storage, Managed Service for Kafka и Compute Cloud.

## Последствия

Инфраструктура описывается через Terraform. Для учебного MVP локально используется MinIO и Redpanda, чтобы проект можно было показать без затрат на облако.
